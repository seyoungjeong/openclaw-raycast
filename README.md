## Setup

1. Install the extension from Raycast Store
2. Configure your OpenClaw settings:
   - **API URL**: Your OpenClaw API endpoint (default: `http://localhost:18789`)
   - **Gateway Token**: Optional authentication token if gateway auth is enabled
   - **Agent Name**: The name of your OpenClaw agent (default: `main`)

## Usage

1. Open Raycast (`âŒ˜ + Space`)
2. Type "Ask OpenClaw" or search for the command
3. Type your question and press Enter
4. View the response and optionally copy it to clipboard

## Requirements

- A running OpenClaw instance
- Access to OpenClaw API (local or remote)
